# Goal Planner (React)

Minimal Goal Planner app scaffolded for planning daily, monthly, and yearly tasks with skill tracking and analytics.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm start
```

- [src/components/TaskList.js](src/components/TaskList.js)
